# 💰 Salary Prediction App Using Machine Learning

## 📌 Project Overview

The Salary Prediction App is a Machine Learning web application developed using Streamlit and Linear Regression. The application predicts an employee's salary based on their years of work experience. Users can enter their experience, and the trained model instantly provides an estimated salary prediction.

## 🎯 Objective

To build an interactive machine learning application that demonstrates how Linear Regression can be used to predict salaries based on experience.

## 🚀 Features

* User-friendly Streamlit interface
* Real-time salary prediction
* Machine Learning model trained using Linear Regression
* Fast and accurate predictions
* Easy deployment and accessibility

## 🛠️ Technologies Used

* Python
* Streamlit
* NumPy
* Scikit-learn
* Pickle
* Linear Regression

## 📂 Project Workflow

1. Collect and preprocess salary dataset.
2. Train a Linear Regression model using years of experience as the input feature.
3. Save the trained model using Pickle.
4. Build an interactive Streamlit web application.
5. Load the saved model and predict salary based on user input.

## 📊 Input

* Years of Experience

## 📈 Output

* Predicted Salary

## ▶️ How to Run the Project

### Install Required Libraries

```bash
pip install streamlit numpy scikit-learn
```

### Run the Application

```bash
streamlit run app.py
```
